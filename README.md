# circles packer
This program can quickly solve the 2D bin packing problem for any number of any circles inside any other polygon! Based on Flamethowers polygon packer.
<img width="640" height="480" alt="30 circles in a hexagon" src="https://github.com/SummerShelf/polygon-packer/blob/00bc463ca08402ac023de3ee3a41726ac786b4ac/13_in_6.png" />

To use it, download the python file, navigate to its location and run it like this:

`python3 polygon_packer.py [n] [nsc]`
- Replace `[n]` with the number of inner circles you want to solve for
- Replace `[nsc]` with the number of sides of the container polygon
